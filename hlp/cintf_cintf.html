Description

Constructors of the CInterface class.

Parameters

First three of function parameters are for initializing of CNode.
ucBR Baud rate for the interface. Possible values are defined as constants in dnmdef.h:
  • DEVICENET_BAUD_125K - for 125 000 bytes per second
  • DEVICENET_BAUD_250K - for 250 000 bytes per second
  • DEVICENET_BAUD_500K - for 500 000 bytes per second

Remarks

CInterface doesn't support copy constructor and assignment operator. They are declared in section private and are not defined.
