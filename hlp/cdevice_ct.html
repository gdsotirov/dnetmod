CDevice::SetConnType, CDevice::GetConnType

Declarations

inline unsigned char CDevice::GetConnType(void) const;
void CDevice::SetConnType(unsigned char ucCT);

Description

This functions can be used get/set the I/O connection type established to device with Allocate function.

Parameters

ucCT Connection type to be set. Possible values for this parameter are:
  • DEVICENET_CONN_POLLED - 0x0001
  • DEVICENET_CONN_STRBED - 0x0002
  • DEVICENET_CONN_COS - 0x0004
  • DEVICENET_CONN_CYCLIC - 0x0008
These constants are defined in dnmdef.h file.

Return values

Connection type for device.

Remarks

Connection type can be set only if device is not active on the network - not allocated.
