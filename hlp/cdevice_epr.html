CDevice::GetEPR, CDevice::SetEPR

Declarations

inline unsigned short CDevice::GetEPR(void) const;
void CDevice::SetEPR(unsigned short usEPR_);

Description

This function can be used to get/set the EPR for the I/O connection established to device.

Parameters

usEPR_ EPR for device.

Return values

EPR for device.

Remarks

EPR can be set only when device is not active on the network - not allocated.
