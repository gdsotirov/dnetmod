CIdentificator

Declaration

class DNETMOD_API CIdentificator

Hierarchy

Base class.

Description

This class is used as a base for all other classes in the project. It provides general identification mechanism for his descendants. A descendant of CIdentificator can identify itself with a single unsigned decimal number or with a name.

Construction/Destruction

Remarks

This class doesn't support copy constructor and assignment operator. They are declared in the private section but are not defined. Check class members to see more information about inheriting this class.
