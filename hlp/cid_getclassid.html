CIdentificator::GetClassID

Declaration

unsigned long GetClassID(void) const;

Description

Retrieves class identification number (ClassID).

Return values

Class identification number.

Remarks

For the identification, two members of the class named ClassID and ClassName are used. This members are static. Thus they are the same for all objects of type CIdentificator and it descendants. If you want to identify differently a descendant of CIdentificator you must declare once again this members and to initialize them like you want. Declaration of the members follow:

protected:
    static unsigned long ulClassID;
    static char strClassName[];
