CDevice::CDevice

Declaration

CDevice::CDevice(
    unsigned char  ucMID,
    unsigned char  ucCCS,
    unsigned char  ucPCS,
    unsigned char  ucCT,
    unsigned short usEPR_,
    CInterface     *pIntf)

Description

Constructor.

Parameters

First three of function parameters are for initializing of CNode
ucCT Type of the I/O connection to device. Possible values for this parameter are:
  • DEVICENET_CONN_POLLED
  • DEVICENET_CONN_STRBED
  • DEVICENET_CONN_COS
  • DEVICENET_CONN_CYCLIC
These constants are defined in dnmdefs.h file.
usEPR_ Expected Packed Rate from device.
pIntf Pointer to a object of type CInterface.
