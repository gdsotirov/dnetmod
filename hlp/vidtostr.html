VendIdToStr

Declaration

DNETMOD_API int DNETMOD_CC
VendIdToString(unsigned short usVendId, unsigned long ulStrSz, char *strName);

Description

This function translates between vendor identificator and vendor name. Vendor names are taken form ODVA web site and are updated before build of the module. So any newly registered vendors after this date will not be present.

Parameters

usVendId Vendor identificator.
ulStrSz Size of the C type string passed as next parameter.
strName Pointer to a C type string where to copy vendor name.

Return values

ERR_NOERROR 0 = successful translation made.
ERR_NOSUCH_VENDID When function didn't find the vendor id passed.

Requirements

#include "dnetmod.h"

OS\Libraries Static (CT) Import library (CT) Shared (RT)
Linux libdnetmod.a - libdnetmod.so.0.2
Win32 dnetmod.lib dnetmodimp.lib dnetmod.dll

Sample code

This code show how all vendor names can be extracted:

...
/* Note: DNETMOD_MAX_VENDSTR_LEN and DNETMOD_MAX_VENDID
         are defined in dnetmod.h */
char vname[DNETMOD_MAX_VENDSTR_LEN];
...
printf("DeviceNet Vendors:\n");
for( int i=0; i<DNETMOD_MAX_VENDID; i++ )
    if( VendIdToStr(i, DNETMOD_MAX_VENDSTR_LEN, vname) == ERR_NOERR )
        printf("%d - %s\n", i, vname);
    else /* show error */;
