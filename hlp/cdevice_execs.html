CDevice::ExecService

Declaration

virtual int ExecService(
    unsigned char  ucSrvCode,
    unsigned short usClsId,
    unsigned short usInstId,
    unsigned short usDataSz,
    void           *pvData) = 0;

Description

Function description.

Parameters

ucSrvCode Service code of the service to be executed.
usClsId Class identificator for the service.
ucInstId Instance identificator for the service.
usDataSz Length of the service data in bytes.
pvData Service data buffer.

Return values

Function must return an error code and to format an error message when error detected in the process of executing service in device. Also if device responds with an error on the service request it's good to include error codes in the error message.

When the user invokes this function it must pass a pointer to a buffer with data for the service. After function return in this buffer must have data returned from the service if any.
