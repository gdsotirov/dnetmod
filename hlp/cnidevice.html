CNIDevice

Declaration

class DNETMOD_API CNIDevice : public CDevice

Hierarchy

Description

This class is a shell for a slave device attached to NationalInstruments NI-DNET™ interface. It is designed to be used with CNIInterface class and the NI-DNET™ driver. This class is only available on Win32 platform.

Class members

Construction/Destruction
CNIDevice Constructor. pIntf parameter must be of class CNIInterface.
~CNIDevice Destructor. It will invoke Unallocate method in case the user forgot.
