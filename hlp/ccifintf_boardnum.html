CCIFInterface::GetBoardNum, CCIFInterface::SetBoardNum

Declaration

inline unsigned short GetBoardNum(void) const;
void SetBoardNum(unsigned short usBrdNum);

Description

This function can be used to retrieve or set number of the Hilscher board to be used.

Parameters

usBrdNum Hilscher board number. Must be a value between 0 and 3.

Return values

Hilscher board number.
