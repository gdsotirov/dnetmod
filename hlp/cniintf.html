CNIInterface

Declaration

class DNETMOD_API CNIInterface : public CInterface

Hierarchy

Description

This class is a shell for NationalInstruments NI-DNET™ interface. It is based on CInterface class and is available only on Win32 platform, because the C API of the interface driver is only for this platform.
