CNIInterface::GetIntfID, CNIInterface::SetIntfID

Declarations

unsigned char GetIntfID(void) const;
void SetIntfID(unsigned char ucIID);

Description

This functions can be used to retrieve or set NI-DNET™ interface identificator.

Parameters

ucIID NI-DNET™ interface identificator. Must be a number between 0 and 31.

Return values

NI-DNET™ interface identificator.
