CDevice::GetAttribute

Declaration

virtual int GetAttribute(
    unsigned short usClsId,
    unsigned short usInstId,
    unsigned char  ucAttrId,
    unsigned short usDataSz,
    void           *pvData,
    unsigned short *pusActDataSz) = 0;

Description

This function must allow the user to retrieve device's attribute. User must provide buffer with appropriate size for the attribute data.

Parameters

usClsId Class identificator of the attribute.
usInstId Instance identificator of the attribute.
ucAttrId Attribute identificator.
usDataSz Length in bytes of the attribute data.
pvData Pointer to a buffer where to copy attribute data.
pusActDataSz Actual size of the attribute data in bytes.

Return values

Function must return an error code and to format an error message when error detected in process of retrieving device's attribute. Function must not report as an error inconsistency between usDataSz and actual size of the attribute data. Actual length of the attribute data in bytes must be fixed in pusActDataSz.
