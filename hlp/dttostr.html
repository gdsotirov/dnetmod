DevTypeToString

Declaration

DNETMOD_API int DNETMOD_CC
DevTypeToString(unsigned short usDvType, unsigned long ulStrSz, char *strName);

Description

This function translates between corresponding device type identificator and device type string. Device type strings are taken from DeviceNet specification.

Parameters

usDvType Device type identificator.
ulStrSz Size of the C type string you pass as next parameter.
strName The address of a C type string where to copy the according device type name.

Return values

ERR_NOERROR 0 = successful translation made.
ERR_NOSUCH_DEVTYPE When function didn't find the device type id passed.

Requirements

#include "dnetmod.h"

OS\Libraries Static (CT) Import library (CT) Shared (RT)
Linux libdnetmod.a - libdnetmod.so.0.2
Win32 dnetmod.lib dnetmodimp.lib dnetmod.dll

Sample code

This code shows a possible use of the function.

...
unsigned short id = 1;
/* Note: DNETMOD_MAX_DTSTR_LEN is defined in stdata.h */
char dtstr[DNETMOD_MAX_DTSTR_LEN];
...
if( DevTypeToString(id, DNETMOD_MAX_DTSTR_LEN, dtstr) == ERR_NOERR )
    printf("ID=%d, Str=%s", id, dtstr); 
    /* will produce: ->ID=1, Str=Generic Device */
else /* function can't make translation */;
