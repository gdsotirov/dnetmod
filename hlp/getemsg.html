GetErrMsg

Declaration

DNETMOD_API void DNETMOD_CC
GetErrMsg(unsigned long ulStrSz, char *strMsg);

Description

Use this function to retrieve descriptive message for the last error detected.

Parameters

ulStrSz Size of the C type string you pass as next parameter.
strMsg Pointer to a C type string where to copy error message.
