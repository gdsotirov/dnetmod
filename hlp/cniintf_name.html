CNIInterface::GetName

Declaration

void GetName(unsigned long ulStrSz, char *strName);

Description

This function can be used to retrieve NI-DNET™ interface name. Name is a string with format "dnetxx", where xx is the interface identificator. So valid names are dnet1, dnet2, dnet3, ... dnet31.

Parameters

ulStrSz Size in bytes of the string passed as next parameter.
strName Pointer to a string where to copy interface name.
