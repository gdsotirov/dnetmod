CNode::GetConsumedConnSize, CNode::SetConsumedConnSize

Declarations

inline unsigned char CNode::GetConsumedConnSize(void) const;
void CNode::SetConsumedConnSize(unsigned char ucCCS);

Description

Use this functions to get and set count of node consumed bytes from the network.

Parameters

ucCCS Node consumed data bytes from the network.

Return values

Node consumed data bytes from the network.

Remarks

Function SetConsumedConnSize will set the new value only if node is not active.
