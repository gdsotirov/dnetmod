CDevice::SetAttribute

Declaration

virtual int SetAttribute(
    unsigned short usClsId,
    unsigned short usInstId,
    unsigned char  ucAttrId,
    unsigned short usDataSz,
    void           *pvData) = 0;

Description

This function must allow the user to set device's attribute.

Parameters

usClsId Class identificator of the parameter.
usInstId Instance identificator of the parameter.
ucAtrrId Parameter identificator.
usDataSz Length of the parameter data in bytes.
pvData Pointer to a buffer where to copy parameter data.

Return values

Function must return a value and to format an error message when error detected in the process of setting device's attribute. Error must be returned if usDataSz is less or greater than actual size of attribute data.
